# PRODIGY_WD_02 — Stopwatch Web Application

## Task
Build a stopwatch web app using HTML, CSS, and JavaScript with start, pause, reset, and lap-time tracking functionality.

## Tech Used
- HTML5
- CSS3
- Vanilla JavaScript (Date.now() + setInterval for accurate timing)

## Features
- Start / Pause / Resume / Reset controls
- Lap time recording with a scrollable lap list
- Centisecond-accurate display (HH:MM:SS.CS)
- Fully responsive single-card layout

## How to Run
1. Clone this repo
2. Open `index.html` in any browser
